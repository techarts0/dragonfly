## 介绍
Dragonfly是一个轻量级的Java应用开发框架，它基于DI框架[Whale](https://gitee.com/techarts/whale)，包括：
-A. 一个RESTful风格的Web Framework，以JSR371和JAX-RS标准为参考；
-B. 一个以统一的接口使用MyBatis, Apache DBUtils和Apache OPENJPA的Data Framework.
